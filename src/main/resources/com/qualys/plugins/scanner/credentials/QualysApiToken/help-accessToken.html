Enter your Qualys API access token.

How to obtain an access token:

  1. Log in to your Qualys platform
  2. Navigate to Administration > Users > API Access
  3. Generate a new API token or use an existing one

Security:

  • The token is stored encrypted in Jenkins credentials store
  • The token is passed to QScanner via environment variable (QUALYS_ACCESS_TOKEN)
  • The token is never logged or exposed in command line arguments

Best practices:

  • Use a dedicated service account token for CI/CD
  • Rotate tokens periodically
  • Use the minimum required permissions
